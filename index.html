Proportional Scaling

Components scale according to:

  • font-size
  • viewport

Relative units

  • em
  • ch
  • %
button {
  font-size: 1em;
  padding: 0.5em 0.75em;
  /* Other properties */
}

/* Group common properties together */
.button {
	display: inline-block;
  padding: 0.5em 0.75em;
  font-size: 1em;
}

.button-small {
  font-size: 0.8em;
}

.button-large {
  font-size: 2em;
}

Using Sass mixins

@mixin button {
	display: inline-block;
  padding: 0.5em 0.75em;
  /* Other common properties */
}

.button-large {
  @include button;
  font-size: 2em;
}

Responsive Scaling

Media queries

Media queries

  • min-width queries
  • max-width queries

min-width media queries

// Note: Always write media queries in `em`.
.component {
  @media (min-width: 37.5em) {
    font-size: 2em;
  }
}

max-width media queries

.component {
  font-size: 2em;
  @media (max-width: 37.5em) {
    font-size: 1em;
  }
}
